writeLongLe

fun writeLongLe(v: Long): Buffer

Overrides BufferedSink.writeLongLe

Writes a little-endian long to this sink using eight bytes.

buffer.writeLongLe(9223372036854775807L);
buffer.writeLongLe(15);
